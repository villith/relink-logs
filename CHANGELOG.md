# Changelog

Hand-written notes for each release. The release workflow refuses to tag a
version without a section here, and the section body becomes the GitHub
release body — which the in-app update prompt shows as patch notes. Keep
entries as short plain-text bullets; the prompt renders them without markdown.

## 1.10.0

- New Toolbox tool: Overmastery Predictor — preview a character's upcoming overmastery rolls and spot the roll that hits your wanted traits
- The update prompt now shows patch notes, and a new setting can turn automatic update checks off
- Synthesis Helper remembers your search between sessions
- Toolbox inputs pause while data is being read from the game
- Saving an encounter no longer navigates away while you are using the Toolbox
- Stun/s shows two decimal places
